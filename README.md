# Django-CRUD-API-Project




## Django REST CRUD API with MongoDB
This is a simple CRUD API project built using Django REST Framework and MongoDB as a database backend. The API provides endpoints for creating, reading, updating, and deleting resources in the database.



## Technologies Used
***Python***

***Django***

***Django RESTFramework***

***MongoDB***


## Setup

Clone the repository to your local machine.

Make sure you have Python and pip installed on your system.

Create a virtual environment and activate it.

Install the dependencies using pip install -r requirements.txt.

Start the Django development server using python manage.py runserver.

You can now make requests to the API endpoints at http://localhost:8000/.
